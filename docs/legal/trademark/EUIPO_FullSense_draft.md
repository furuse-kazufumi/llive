## 4. Filing strategy

Three main options for EU-coverage:

| Option                              | Pros                           | Cons                          |
|-------------------------------------|--------------------------------|-------------------------------|
| **EUTM** (direct EUIPO)             | Single application, 27 states  | Single-state opposition kills | 
| **Madrid Protocol from JP base**    | Cheap if filing several countries | JP base must register first |
| **National TM in each EU country**  | Survives single-state issues   | Expensive, slow               |

**Recommended**: EUTM (direct EUIPO) as the primary EU filing. Combine
with Madrid Protocol from JP base for non-EU jurisdictions later.

## 7. Filing process

  1. Pre-search via eSearch plus + TMview
  2. Create account at EUIPO website (`https://euipo.europa.eu/`)
  3. Online application via "Apply for a trade mark"
  4. Enter applicant, mark, classes, goods/services
  5. Pay filing fee (EU bank transfer / credit card / current account)
  6. Receive filing number
  7. Examination (~3-6 months) — formalities + absolute grounds
  8. Publication for opposition (3 months)
  9. Registration certificate issued

## 8. Languages

EUIPO accepts filings in any of the 24 EU official languages, but
correspondence is conducted in one of 5 working languages: EN / DE / FR /
ES / IT. **English** is recommended for a non-EU applicant.

A second language must be designated for opposition / cancellation
procedures (must be different from the first language).
