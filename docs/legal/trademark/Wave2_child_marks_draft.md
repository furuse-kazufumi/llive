## Pre-filing checklist (per mark, per jurisdiction)

For each combination (3 marks × 3 jurisdictions = 9 cases):

- [ ] J-PlatPat search: 商標称呼検索 (called names) + 文字列 + 図形
- [ ] USPTO TESS search: word mark + design
- [ ] EUIPO eSearch + TMview: word mark
- [ ] Confirm no live registration in Class 9 / 42
- [ ] Confirm no pending application with earlier priority
- [ ] Domain availability check (`<mark>.com`, `.ai`, `.dev`, `.io`)
- [ ] Designate US attorney for non-US applicant (USPTO only)
- [ ] Prepare specimens (PyPI page + GitHub README screenshots)

## Estimated total cost (self-filing, 2 classes per mark)

| Mark   | JP (出願+登録)         | US (TEAS Plus 2 classes) | EU (EUTM 2 classes)   |
|--------|------------------------|--------------------------|-----------------------|
| llmesh | ¥24,000 + ¥65,800      | $500                     | €900                  |
| llive  | ¥24,000 + ¥65,800      | $500                     | €900                  |
| llove  | ¥24,000 + ¥65,800      | $500                     | €900                  |
| **合計** | **¥269,400**         | **$1,500**               | **€2,700**            |

Grand total: roughly **JPY 270k + USD 1.5k + EUR 2.7k ≒ JPY 720k** for
self-filing (excluding attorney fees). With Wave 1 (FullSense) added, the
overall budget for trademark protection across 4 marks × 3 jurisdictions is
approximately **JPY 1.0M** self-filed, or **JPY 2.0–2.5M** with attorney
representation.
